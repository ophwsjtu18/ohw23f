### 1. 产品概述

陆地游泳器，是一种颠覆性的运动工具，产品采用最尖端的虚拟现实技术和气垫浮力系统，让您体验前所未有的陆地游泳之旅。

### 2. 技术规格

| 技术规格     | 描述             |
| ------------ | ---------------- |
| 游泳速度     | 1米/100秒        |
| 气垫浮力系统 | 永不下沉         |
| 备用氧气罐   | 用于提供清新空气 |

### 3. 特色功能

- 欣赏沿岸美丽的室内景观。
- 定制海水口味。
